MultiFOLD – Results Help Guide

Use this guide to interpret prediction confidence, colouring, and scores.

Key Information

Show Chains

Displays the number of chains in the predicted structure.

Show Quality

Colours the structure by prediction quality:

  • Blue: Higher confidence
  • Red: Lower confidence

Score Meanings

Most scores range from 0 to 1. Values closer to 1 indicate higher confidence. Error-based scores are interpreted oppositely (lower is better).

Assembly Score

Confidence in overall chain assembly.

Interface Score

Confidence in predicted interfaces.

ModFOLDIA

Overall model quality score.

pLDDT

  • 90–100: Very high
  • 70–90: High
  • 50–70: Medium
  • <50: Low

pTM

Global structure confidence.

ipTM

Interface positioning confidence.

pDockQ / pDockQ2

Docking/interface quality.

Cons_IF / Cons_fold / Cons_total

Consensus agreement scores.

globalIF

Global interface quality.

VoroMQA

Structure quality based on atomic packing.

CDA

Contact-based accuracy score.

LDDTjury / CADjury / QSjury / DockQjury

Consensus scores for structure and interfaces.

GDT_TSjury / TMjury

Global structural similarity scores.

VoroIFglobal / VoroIFlocal

Interface quality scores.

ICSjury / IPSjury / iLDDTjury

Interface consensus scores.

ipSAE / ipSAE_d0chn / ipSAE_d0dom

Error-based interface scores (lower is better).

Missing scores

A dash indicates the score was not available.

FAQ

What does a high pTM score mean?

A pTM score ranges from 0 to 1. Values closer to 1 indicate higher confidence.

Why are some regions red?

Red regions indicate low confidence predictions.

How should Assembly Score be interpreted?

Higher values indicate more reliable chain arrangement.
